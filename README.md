## 📱 About

Cal-Track is an innovative **Progressive Web Application (PWA)** that revolutionizes how you track nutrition for Indian cuisine. Simply take a photo of any Indian meal, and our advanced AI engine will provide comprehensive nutritional analysis including calories, proteins, carbohydrates, fats, and fiber content.

## 🚀 Getting Started

### Prerequisites

Ensure you have the following installed:
- **Node.js** (v14 or higher)
- **Python** (v3.8 or higher)
- **Git**

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/Prajwal-koundinya/Cal-track-mobile-app.git
cd Cal-track-mobile-app
```

2. **Backend Setup**
```bash
cd backend
pip install -r requirements.txt
python app.py
```

3. **Frontend Setup**
```bash
cd frontend
npm install
npm start
```

4. **Access the Application**
- Open your browser and navigate to `http://localhost:3000`
- For mobile testing, use your device's browser with your local IP

## 🎯 How It Works

1. **📸 Capture**: Take a photo of your Indian meal using the app's camera interface
2. **🤖 Analyze**: AI processes the image to identify dishes and estimate portions
3. **📊 Report**: Receive detailed nutritional breakdown with calorie and macro information
4. **📈 Track**: Monitor your daily nutrition goals and protein requirements
5. **🎯 Optimize**: Get personalized recommendations based on your eating patterns
